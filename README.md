For python 2, use the following in `poi_id.py`:

```

```

For python 2, use the following in `poi_id.py`:

```

```

## Generating result from pkl files

Assuming the pkl files are in project_dir:

```
tester.py project_dir
```

or

```
tester3.py project_dir
```
